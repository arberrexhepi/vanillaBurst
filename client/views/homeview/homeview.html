What is vanillaBurst?

  • A standalone or plugin solution.
  • Ensures your required files are ready for the function, DOM, and server you are interacting with.
  • Control the behavior of an existing JS project or create your own entirely!
  • Builds and prepares the project as [vanillaApp].
  • No additional helper functions needed if all you need is JS files promised exactly where you need them.
  • Never wonder why your clicks aren't firing again, especially in something like WordPress!

What are vanillaScoops?

Scoops are essentially plugins or components. A Scoop is any combination of a landing, components and function filed that can be added as a package to any number of landings. These can be visual or functional. There's many throughout this site!

What else can vanillaBurst do?

  • Packs and load balances a project's files by endpoint i.e: ?burst=home.
  • Has the capability for state caching and state interaction via server calls and signals.
  • The simplified server communication [helper function "serverRender"] alongside the [helper function "myBurst"] allows for dynamic and quick interaction and availability.
